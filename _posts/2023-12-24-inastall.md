## 3. React

#### 3.1. Node.js
#### 3.2. yarn
- npm install -g yarn
- yarn add [설치할 패키지 이름]
#### 3.3 CRA 프로젝트 생성
- cd 폴더이름
- yarn create react-app week-1
#### 3.4 프로젝트 실행
- cd 폴더이름
- yarn start
#### 3.5 절대경로 만들기
src 내의 모든 파일은 전부 상위 경로 지정 없이 src에 소속됨
``` javascript
{
	"compilerOptions": {
		"baseUrl": "src"
	},
	"include": ["src"]
}
```
